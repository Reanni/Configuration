# 组件化模块间交互的方式

| 使用方式  |                               缺点                               |
|:--------:|:----------------------------------------------------------------:|
| EventBus |                 EventBean维护成本太高，不好去管理                 |
|   广播   |                      不好管理，都统一发出去了                      |
| 隐士意图  | AndroidManifest.xml里面需要写的配置太多了,维护不方便,只能组件间交互 |
|  类加载  |                    缺点较少 : 容易写错包名类名                     |
| 全局Map  |                    缺点较少 : 要注册很多的对象                     |
